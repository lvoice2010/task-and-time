# Дела — канбан + тайм-трекер

Одностраничное React-приложение: канбан-доска (матрица Эйзенхауэра) + тайм-трекер + аналитика по направлениям и компаниям.

## Запуск локально

Просто открыть `index.html` в браузере. Без сборки — React и Babel standalone подгружаются с CDN.

## Стек

- React 18 (CDN)
- Babel Standalone (in-browser JSX)
- localStorage (ключ `kanban-v4`)
- Тёмная тема, DM Sans + JetBrains Mono
